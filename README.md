# todo-vue

## Todo site prototype
Written as a knowledge test  
Used Vue, Vuex, Vuetify  
Testing by Jest

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests watched mode
```
npm run test:watch
```
